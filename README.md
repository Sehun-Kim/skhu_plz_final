# 성공회대 강의평가 서비스 [SkhuBee](https://skhubee.herokuapp.com/)
성공회대학교 멋쟁이 사자처럼 4기 프로젝트
> by 김세훈, 김은석, 정지원

### Environment
- Rails 4.2.5
- sqlite3
- Devise
- Rolify
- Authority
- SearchCop

### Requirement
1. 이메일 인증을 통한 회원가입 기능
2. 강의별 게시판 기능
3. 강의에 대한 별점 평가 기능
4. 강의명, 교수명 검색 기능
5. 게시글 좋아요 기능
